# Vocabulary

* [Editor](#editor)

## Editor

A piece of software which is able to change the content of a file.
In the case of Redaktilo, the editor is an object provided by a library.

## Text file

To be opposed to "binary file". Can contain:

* plain text
* JSON
* YAML
* PHP
* etc...
